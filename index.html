Welcome to WebRTC Screen Sharing

This project was developed as a university project for Multimedia Technology. It was created as a presentation and contains a working demo.

Technologies used

  • Node.js
  • Socket.io
  • WebRTC

Why use WebRTC

Say you want to screen share with someone who is on the same building as you but on a different floor. WebRTC establishes peer-to-peer communication, meaning that your computer is gonna send packages only to the computer that you are trying to share. Data is gonna travel to the first node of the internet and then back making the communication extremely efficient.
